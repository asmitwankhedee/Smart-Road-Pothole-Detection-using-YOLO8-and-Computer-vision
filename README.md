# Pothole-and-Cracks-Detection

A web application using YOLO for detecting potholes and cracks on roads.

## Features
- Upload road images for analysis
- AI-based detection using YOLO models
- Provides exact pixel dimensions of detected potholes and cracks
- Secure OTP login integration

## Setup
1. Clone the repository
2. Install requirements
3. Create a `.env` file for credentials
4. Run `app.py`
